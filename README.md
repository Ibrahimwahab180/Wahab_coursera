# Coursera Data Analysis Projects

A collection of data analysis projects from Coursera courses demonstrating data science skills.

##  Project Overview

This repository contains Jupyter notebook analyses covering:
- Data cleaning and preprocessing
- Exploratory data analysis (EDA)
- Statistical modeling
- Data visualization
- Machine learning applications

##  Technical Skills

- **Languages**: Python
- **Libraries**: Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn
- **Tools**: Jupyter Notebook, Git

##  Files

- `Wahab_coursera2.ipynb` - Main analysis notebook
